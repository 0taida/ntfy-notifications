Troubleshooting Tips

  • VAPID Key Format: Should be 87 characters long (65 bytes encoded in base64)
  • HTTPS Required: Web Push only works on HTTPS or localhost
  • Browser Support: Make sure you're using a modern browser (Chrome, Firefox, Edge)
  • Service Worker: Must be served from the same domain as your web app
  • ntfy Configuration: Ensure your ntfy server has web push enabled with the correct VAPID keys
